Methodology

Pipeline Overview

End-to-end ML pipeline from raw network traffic to explainable predictions.

01

Data Preprocessing

Load CIC-IDS-2017 CSV files, clean infinite/NaN/duplicate values, encode traffic classes, standardize 78 features.

02

Model Training

Train Logistic Regression, Random Forest, and XGBoost with stratified 70/10/20 train/val/test splits.

03

Evaluation

Compute accuracy, precision, recall, F1-score. Generate confusion matrices and model comparison charts.

04

Explainability

SHAP for global feature importance. LIME for local instance-level explanations of predictions.

Limitations & Known Issues

We believe in transparency about what this project can and cannot do.

✓ Real Data Benchmark

Real CIC-IDS-2017 V2 benchmark (100K sample). Random Forest: 99.69% accuracy. Significantly better than synthetic (0.51 macro F1).

⚠ XGBoost Labels

XGBoost has issues with rare classes dropped during stratified split. Random Forest handles this better.

◈ Sample Size

Current benchmark uses 100K sample. Full dataset (2.8M+) would provide more comprehensive evaluation.

✓ What's Working

Cross-validation, PR curves, calibration, failure analysis, Docker, FastAPI, and tests all implemented.
